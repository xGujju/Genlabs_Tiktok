### Research sources checked
- Google NotebookLM Help: supports PDFs, websites, YouTube URLs, audio, Google Docs/Slides, Word/PPTX/CSV/Markdown/ePub and can transform sources into study guides, briefings, audio overviews, mind maps, flashcards/quizzes, infographics, and slide decks.

- Gamma Pricing: free plan exists for simple projects, 400 signup credits, up to 10 cards per prompt, import PDF/PPTX, export PDF/PPTX/PNG/Google Slides.
- ChatGPT capability overview: useful for drafting, rewriting, summarizing, translating, image/document analysis, search/deep research depending on access tier.
- Perplexity Blog/help pages: current product direction emphasizes source-backed answers and research-style workflows; good for buyer research and comparing options.

### Durable content lessons
1. Best mass-audience angle is not “new AI tool list”; it is “ทำงาน X ให้เสร็จใน 10 นาทีด้วยเครื่องมือฟรี”. Thai viewers save/share when the workflow maps to a real deadline: exam, job application, product post, client proposal, video caption, price comparison.
2. Beginner-friendly carousels should show: input → AI action → usable output. Avoid abstract tool descriptions.
3. Strong repeatable Thai audience buckets:
   - Students: summarize lecture/PDF → quiz/flashcards → exam plan.
   - Workers: messy notes/PDF → one-page brief/email/action list.
   - Freelancers: client brief → proposal/slide deck/content samples.
   - Shop owners: product photo + details → listing caption/poster/short video.
   - Small businesses: supplier/competitor links → comparison table + decision checklist.
4. Save/share mechanics: include exact prompts, 3-step recipes, and “copy today” templates. Avoid jargon like RAG, multimodal, agents unless translated into plain Thai benefit.
5. Visual pattern for GenLabs carousel: first slide = painful situation; middle slides = 3-step recipe with tool names; final slide = copyable prompt/checklist CTA.

### Shortlist ideas for Content Pipeline
- NotebookLM study pack: upload class PDFs/YouTube/source links → generate Thai study guide + flashcards/quizzes → ask for 7-day exam plan.
- Canva + ChatGPT/Canva Magic Write shop kit: product details/photo → generate Thai selling points/caption → create poster/story in Canva.
- Gamma client proposal: paste client problem + services → generate 6-slide proposal → export PDF and send to client.
- Gemini/ChatGPT document assistant: upload work PDF or meeting notes → ask for summary, risks, action items, email reply.
- Perplexity buyer-research table: ask for comparison with sources → make 3-option buying checklist for shop owners/freelancers.
- CapCut subtitle rescue: record raw phone video → auto captions → cut into 15-30 sec useful clip.

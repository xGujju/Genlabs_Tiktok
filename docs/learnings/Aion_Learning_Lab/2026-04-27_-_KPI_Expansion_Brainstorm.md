## Expansion Lanes Beyond Static Carousel
### 1. Narrated carousel videos
Turn the same 5-slide carousel into a short voiceover video with simple pan/zoom motion.
- Low extra production cost.
- Higher retention potential than static slideshow.
- Thai narration can build brand personality.

### 2. Screen-record tutorials
30–90 second “do this with me” AI workflows.
Examples:
- make a product photo with AI
- create a customer reply bot
- summarize PDFs
- build a small landing page with Claude Code
- automate a spreadsheet task

### 3. Prompt pack posts
Each post teaches one copyable prompt and one use case.
CTA: save this / comment “prompt” / download free pack.

### 4. Mistake/fix format
Hook: “คนส่วนใหญ่ใช้ AI ผิดตรงนี้…”
Structure: wrong way → why it fails → better prompt/workflow → result.
This is likely broader and more viral than pure news.

### 5. AI tool battle / chooser maps
Simple “ใช้ตัวไหนดี?” posts for beginners.
Examples:
- ChatGPT vs Claude vs Gemini for students/business/design/coding
- free AI tools for shop owners
- best AI tool for Thai content creation

### 6. Before/after transformations
Show concrete transformation:
- bad product photo → AI-enhanced product ad
- messy notes → clean summary/action plan
- boring caption → high-converting TikTok caption
- idea → landing page

### 7. Mini-series format
Repeatable branded sequences:
- AI Skill in 60 Seconds
- AI Tool of the Day
- Prompt You Can Steal
- AI for Thai Small Business
- Claude Code for Non-Coders
- 1 Workflow That Saves 1 Hour

### 8. Free resource funnel content
Posts that naturally lead to free downloads:
- “แจก prompt pack”
- “AI checklist for small business”
- “Claude Code starter workflow”
- “TikTok carousel prompt template”

## Immediate Hypotheses to Test
1. Practical workflow posts will outperform generic AI news.
2. “Mistake → fix” hooks will beat neutral explainer hooks.
3. Beginner-friendly “which AI tool should I use?” posts will get saves/shares.
4. Narrated carousel videos can reuse carousel assets and improve reach without rebuilding production from zero.
5. Claude Code/agent workflow content can build authority, but needs Thai beginner framing to avoid niche jargon.

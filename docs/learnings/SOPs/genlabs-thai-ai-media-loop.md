# GenLabs Thai AI Media Loop

## Purpose
Run the Thai AI media business as a learning system instead of a one-way content factory.

## System Loop
1. Discover signal
2. Rank ideas
3. Rewrite into Thai value
4. Generate carousel assets
5. Publish
6. Collect analytics
7. Review winners and losers
8. Update future selection and formatting rules

## Step 1 — Discover signal
Capture into `#ai-signal-radar`:
- AI news that changes user behavior
- prompts people can use immediately
- new tools worth trying
- clear tutorials or frameworks
- ideas spreading fast on X

Reject:
- low-trust hype
- pure technical noise with no Thai audience value
- content that is only impressive to AI insiders

## Step 2 — Rank ideas
Score each candidate on:
- reach potential
- usefulness
- clarity in Thai
- fit with broad Thai audience
- indirect relevance to GenLabs trust

## Step 3 — Rewrite into Thai value
Every draft should answer:
- what happened?
- why should a normal Thai user care?
- what can they do with this now?
- what simple takeaway belongs in the final slide?

## Step 4 — Generate assets
For launch phase:
- use handwritten Thai carousel/slideshow format
- optimize for first-slide hook clarity
- keep each slide simple and scannable
- avoid clutter and insider jargon

## Step 5 — Publish
Early rule:
- do not fully automate publishing until quality thresholds are clear
- publish manually or with approval gate first

## Step 6 — Collect analytics
Track:
- views
- follower growth
- saves
- shares
- profile visits
- clicks to free resources

## Step 7 — Review performance
At least once per review cycle, ask:
- which topics got reach?
- which hooks got follows?
- which posts underperformed despite strong ideas?
- which visual patterns are working?
- which resource CTAs helped trust without hurting reach?

## Step 8 — Update rules
Promote winning patterns into durable rules:
- hook styles
- topic categories
- visual structures
- CTA placement
- posting cadence

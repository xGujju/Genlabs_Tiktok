# GenLabs Service Industry Creative Lane

## Purpose
Prevent GenLabs AI Learning from repeating generic AI topics by shifting ideation toward Thailand service-based industries.

## Operating rule
Use this uniqueness key before any new batch goes to FAL/Zernio:

`industry + service pain point + AI tool/integration + workflow + visual metaphor`

A tool may repeat across posts, but the industry, customer touchpoint, input/output, risk, prompt, and visual treatment must be specific to that service business.

## Hard rejection rules
- Do not repeat the same industry inside one batch unless Sway explicitly asks for a multi-post series.
- Do not let generic meeting/action copy appear in non-meeting posts, especially `decision / owner / due date / risk`.
- Do not use generic prompt-context formulas as filler.
- Do not count API-safe/Zernio-safe drafts as quality-ready if the source topic, industry, slides, caption, or visual metaphor drift.
- If a V2 replacement exists, V1 must be deleted from Zernio if API-supported or marked superseded/not-visible in Airtable/reporting.

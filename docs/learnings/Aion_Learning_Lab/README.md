# Aion Learning Lab

## Purpose
Aion's self-learning knowledge base for GenLabs AI Learning growth.

This folder stores research from YouTube, search, social platforms, Apify/X scans, creator breakdowns, and performance reviews so Aion improves instead of repeating mistakes.

## North Star
Make GenLabs AI Learning grow through useful, practical Thai AI education that earns views, followers, trust, and eventually revenue for GenLabs.

## Research Sources
- YouTube: creator strategy, TikTok growth, AI tool tutorials, automation workflows, Claude Code/agentic coding tutorials.
- Search/web: TikTok algorithm notes, carousel/slideshow tactics, content marketing research, platform updates.
- Apify/X: viral AI posts, official announcements, creator-format signals, comments/resources/prompts.
- Internal analytics: TikTok/Zernio/Airtable performance results.

## Learning Loop
1. Collect useful source material.
2. Extract repeatable patterns.
3. Convert patterns into GenLabs experiments.
4. Track performance.
5. Promote winners into SOPs and content templates.
6. Kill or revise weak formats quickly.

# GenLabs Thai AI Media Engine Plan

## Goal
Build a Thai AI media operating system that consistently turns AI signal into carousel content, learns from TikTok analytics, and grows a large trust-based audience that later supports GenLabs.

## Architecture
The system will run in layers: discovery, ranking, Thai adaptation, asset generation, publishing, analytics, and improvement. Launch should stay carousel-first with strong quality control, then expand into video and motion only after the first content loop is stable.

## Phases
### Phase 1 — Operating base
- Create project home in Obsidian
- Create Discord operator structure
- Define channel descriptions and pinned rules
- Define the source categories and content pillars
- Define the first KPI dashboard

### Phase 2 — Manual winning loop
- Collect AI signal manually/semi-manually
- Create first batch of Thai carousel posts
- Publish and track results
- Identify first winning hook/topic patterns
- Build first free resource on GenLabs

### Phase 3 — Structured automation
- Automate source collection
- Automate ranking assistance
- Automate Thai draft generation
- Automate visual asset drafting
- Keep publishing behind a quality gate

### Phase 4 — Analytics intelligence
- Connect TikTok analytics
- Compare top vs weak posts
- Build a content-learning rule system
- Improve topic selection and formatting automatically

### Phase 5 — Media expansion
- Add short video explainers
- Add motion-graphic workflows
- Reuse winning carousel ideas in new formats

## Immediate Task Stack
- [ ] Finalize Discord channel names, topics, and pinned messages.
- [ ] Choose the public TikTok brand name.
- [ ] Define source inputs: X plus any supporting AI news/tool sources.
- [ ] Define a ranking rubric for content worth posting.
- [ ] Define the handwritten Thai carousel style system.
- [ ] Define the first publishing cadence.
- [ ] Define the first analytics review cadence.
- [ ] Define the first free guide/resource to host on GenLabs.
- [ ] Publish the first manual batch before auto-posting exists.

## Success Conditions
- The system can generate repeatable Thai carousel content without chaos.
- Analytics can clearly show what drives views and follows.
- The feedback loop changes future content choices.
- The brand grows without becoming a low-trust repost page.

## Failure Modes to Avoid
- becoming a generic AI news copier
- over-automating before quality is stable
- optimizing too early for direct sales instead of audience growth
- creating content too broad to build GenLabs trust
- creating content too niche to grow
